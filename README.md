# 🍲 Django Recipe App

A full-stack web application built with Python and Django that allows users to create, manage, and share their favorite food recipes. 

## ✨ Features
* **User Authentication:** Secure registration, login, and logout functionality.
* **Protected Routes:** Private dashboards secured using Django's `@login_required`.
* **Full CRUD Functionality:** Users can Create, Read, Update, and Delete their own recipe entries.
* **Media Handling:** Seamlessly upload and store recipe images.
* **Responsive UI:** Clean and modern interface styled with Bootstrap.

## 🛠️ Tech Stack
* **Backend:** Python, Django
* **Frontend:** HTML5, CSS3, Bootstrap
* **Database:** SQLite (Development)
* **Version Control:** Git & GitHub
